**Midtrans Automation Test**
Proyek ini merupakan demonstrasi Automation Testing untuk alur pembelian pada situs demo Midtrans menggunakan Robot Framework

**How to Execute the Code**
Pastikan Anda sudah menginstal Python (saya menggunakan latest version yaitu 3.14.5).
Instalasi Library: Buka terminal/command prompt di dalam folder proyek, lalu jalankan perintah berikut:
pip install -r requirements.txt

**How to Test the Code**
Gunakan perintah berikut di Terminal:
robot TestCases.robot

Setelah pengujian selesai, framework akan langsung otomatis menghasilkan file laporan:
- Report: Ringkasan status pengujian (Pass/Fail).
- Log: Detail langkah-langkah eksekusi untuk debugging.
